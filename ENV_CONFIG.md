## Configuration Validation

### Server-side Environment Check
The application validates all required environment variables at startup. Missing variables will cause the server to fail with clear error messages.

```typescript
// server/_core/env.ts validates:
- DATABASE_URL (required)
- JWT_SECRET (required)
- VITE_APP_ID (required)
- OAUTH_SERVER_URL (required)
- BUILT_IN_FORGE_API_URL (required)

```

## Security Best Practices

### ✅ DO:
- Store server-side secrets in environment variables only
- Rotate API keys regularly
- Use different keys for development and production
- Never commit `.env` files to version control
- Use strong, randomly generated secrets (32+ characters)
- Restrict API key permissions to minimum required scope
- Monitor API usage for unusual activity

### ❌ DON'T:
- Expose server-side secrets to frontend code
- Hardcode secrets in source files
- Share secrets via email or chat
- Use weak or predictable secrets
- Reuse secrets across environments
- Commit `.env` files to git
- Log sensitive values

## Troubleshooting

### "Cannot find module" errors
- Verify all required environment variables are set
- Check `.env` file exists in project root
- Restart development server after changing variables

### "Invalid API key" errors
- Verify API keys are correct and not expired
- Check API key permissions and quotas
- Ensure API key is for correct environment (test vs live)

### "Connection refused" errors
- Verify database URL is correct
- Check database is running and accessible
- Verify network connectivity

### "Unauthorized" errors
- Verify OAuth credentials are correct
- Check OAuth redirect URI matches configuration
- Verify session cookie is being set correctly

## Production Deployment

When deploying to production:

1. **Update Stripe Keys:** Switch from test to live keys
2. **Update OAuth URLs:** Use production domain
3. **Update Database:** Use production database URL
4. **Update API Keys:** Use production API keys
5. **Enable HTTPS:** Ensure SSL/TLS is enabled
6. **Verify Webhooks:** Update webhook URLs to production domain
7. **Test Payment Flow:** Verify Stripe integration works
8. **Monitor Logs:** Watch for configuration errors
